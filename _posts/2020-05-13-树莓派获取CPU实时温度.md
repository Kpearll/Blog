### 树莓派获取CPU实时温度
1. 进入 ```/sys/class/thermal/thermal_zone0``` 目录
``` 
cd /sys/class/thermal/thermal_zone0 
```
2. 查看文件内容
```
cat temp
```
文件的返回值： 
```
51540
```
返回值除以1000为当前CPU温度值，即当前温度为51摄氏度。
